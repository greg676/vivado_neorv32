## Block Design Components

1. **neorv32_vivado_ip_0** — NeoRV32 RISC-V
   - BOOT_MODE_SELECT=0 (internal bootloader), IMEM 32KB, DMEM 8KB
   - I/D caches 4-block, 64B lines, burst mode
   - OCD/JTAG enabled, UART0 enabled, TWI enabled, GPIO (4 in, 3 out)
   - ISA: RV32I + Zicond (no M, no C)
   - ⚠️ DMEM_BASE = 0x80000000 (same as DDR3 MIG!)

2. **clk_wiz_0** — MMCM 100MHz → 100/200MHz for MIG
   - 12.288/24.576MHz outputs defined but disabled

3. **mig_7series_0** — DDR3 controller, 256MB @ 0x80000000

4. **smartconnect_0** — AXI: DMA + NeoRV32 → MIG
5. **smartconnect_1** — AXI: NeoRV32 → DMA regs + path to DDR + TX FIFO + RX FIFO

6. **axi_dma_0** — single-channel DMA, no scatter-gather
7. **axi_fifo_mm_s_0** — TX FIFO (MM2S → sd_out)
8. **axi_fifo_mm_s_1** — RX FIFO (sd_in → S2MM)
9. **sd_in_0** — TDM audio sample IN (NOT FT601!) — custom IP
10. **sd_out_0** — TDM audio sample OUT (NOT FT601!) — custom IP
11. **ilconcat_0 + ilvector_logic_0** — interrupt routing + OCD reset

## Data Flow (TDM Audio)

```
sd_in (TDM RX) → AXIS → fifo_mm_s_1 → DMA S2MM → smartconnect_0 → MIG/DDR3
DDR3 ← smartconnect_0 ← DMA MM2S ← fifo_mm_s_0 ← AXIS ← sd_out (TDM TX)
```

## Segger Embedded Studio Assessment

- **Download**: `Setup_EmbeddedStudio_Linux_x86_64.tar.gz` from segger.com
- **Version**: v8.28 (2026-04-08), unified ARM+RISC-V
- **License**: Free for non-commercial use; commercial needs CUL
- **FPGA bitstream programming**: SES does NOT directly program FPGA bitstreams. That needs Vivado hardware manager or openFPGALoader. J-Link can do SVF-based FPGA programming via JTAGLoad but it's slow and not the primary use case.
  - For Xilinx 7-series, Vivado is still needed for bitstream loading.
  - SES is for firmware development/debug on the NeoRV32 via J-Link.
- **NeoRV32 integration**: SES connects to NeoRV32 OCD via J-Link over JTAG. Uses standard RISC-V debug module.
- **OpenOCD alternative**: NeoRV32 ships with OpenOCD configs (ftdi interface, JTAG). Can be used with GDB or SES.
